### Bagian 3: Tracking Scripts
**`client/index.html`** ÔÇö tambah script placeholder untuk:
- Google Analytics (gtag.js)
- Google Tag Manager (googletagmanager.com)
- Facebook Pixel

Script diaktifkan/dinonaktifkan berdasarkan konfigurasi di settings admin.

**`client/src/components/TrackingScripts.tsx`** ÔÇö komponen yang membaca konfigurasi dari settings API dan menginjeksi script tracking. Dipasang di root App.

### Bagian 4: Settings Admin ÔÇö Tracking
**`client/src/pages/admin/Settings.tsx`** ÔÇö tambah tab/bagian "Tracking & Analytics":
- GA4 Measurement ID (G-XXXXXXXXXX)
- GTM Container ID (GTM-XXXXXXX)
- Facebook Pixel ID
- Semua disimpan ke DB via `payment_gateway` config

### Bagian 5: UTM Capture + Conversion Tracking
- Saat halaman package dibuka, tangkap UTM params dari URL (utm_source, utm_medium, utm_campaign, utm_term, utm_content)
- Simpan ke `sessionStorage`
- Saat checkout (POST /transactions), kirim UTM data ke backend
- `transactions` table: tambah kolom `utm_source`, `utm_medium`, `utm_campaign` (nullable)
- Conversion tracking: saat pembayaran sukses, trigger `gtag('event', 'purchase', {...})`
